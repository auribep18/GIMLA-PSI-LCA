# GIMLA PSI LCA

Welcome to the GIMLA PSI LCA project! This project aims to provide comprehensive life cycle assessments for various processes and systems using the GIMLA methodology.

## Features
- Detailed analysis of environmental impacts.
- User-friendly interface for data input and visualization.
- Customizable options for different assessment scenarios.
